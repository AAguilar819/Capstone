Emergency Disaster Aversion

Emergency Disaster Aversion is a non-profit organization that was created with the purpose to help countries that are in need by sending boxes of survival supplies and some ways to signal for help. Due to the fact that we have only recently became an organization, funding's tight with us, and we'd appreciate if you could donate to our cause. Even a dollar can go a long way, and we intend to put it to good use by making our location more effective at packing the boxes we send so that more can be made for the nations that have a crisis. If we can recieve enough donations, it may be possible to expand our organization to be able to improve what we can send, how much we can produce to assist, and allow for more locations for volunteers to gather at. Eventually, we may find the time to even sell the supplies we create back to the volunteers to help gather funds to assist the nations that need help.
